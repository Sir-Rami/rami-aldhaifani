Google Site Verification

This file is used to verify ownership of this website with Google Search Console.

Replace the meta tag in the main index.html file with your actual Google verification code when you set up Google Search Console.

How to verify your site with Google:

  1. Go to Google Search Console
  2. Add your website URL
  3. Choose "HTML tag" verification method
  4. Copy the verification meta tag
  5. Add it to the <head> section of your index.html file
  6. Click "Verify" in Google Search Console
